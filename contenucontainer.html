Flex sur un container

Display

Display:flex permet "d'activer" le mode flex sur un container. Le container peut être tout type d'élément, que ce soit div, article, ul, li, etc.

.container {
display: flex /*ou*/ inline-flex /*active flex sur le container et le met en inline*/;
}

Flex-direction

Cette propriété que l'on applique au flex-container permet de choisir sa direction, en colonne (vertical) ou en rang (horizontal), en sens normal (de gauche à droite et de haut en bas) ou inversé (de droite à gauche et de bas en haut).

.container {
flex-direction: row /*ou*/ row-reverse /*ou*/ column /*ou*/ column-reverse;
}

Flex-wrap

Flex-wrap permet d'indiquer au conteneur si les items qu'il contient doivent automatiquement aller à la ligne ou non, dans le sens normal (les items qui débordent iront se placer en dessous) ou inversé (les items se placeront au dessus).

.container {
flex-wrap: wrap /*ou*/ no-wrap /*ou*/ wrap-reverse;
}

Flex-flow

Flex-flow permet de déclarer en une seule ligne les valeurs de flex-direction et flex-wrap dans le CSS.

.container {
flex-direction: valeur de flex-direction /*et*/ valeur de flex-wrap;
}

Justify-content

Cette propriété permet de modifier le positionnement vertical des items, mais aussi leur répartition dans l'espace du container.

.container {
justify-content: flex-start /*ou*/ flex-end /*ou*/ center /*ou*/ space-around /*ou*/ space-between;
}

Align-items

Cette propriété permet de modifier le positionnement horizontal des items, mais aussi leur répartition dans l'espace du container.

.container {
align-items: flex-start /*ou*/ flex-end /*ou*/ center /*ou*/ baseline /*ou*/ stretch;
}

Align-content

Quand il y a plusieurs lignes d'items dans le container, align-content permet de distribuer l'espace restant entre ces différentes lignes d'items.

.container {
align-content: flex-start /*ou*/ flex-end /*ou*/ center /*ou*/ space-around /*ou*/ space-between /*ou*/ stretch;
}
