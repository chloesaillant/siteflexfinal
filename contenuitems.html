Flex-grow

Permet de définir la capacité pour un élément à s'étirer dans l'espace restant après distribution entre chaque item.

Flex-shrink

Permet de définir la capacité pour un élément à se contracter si nécessaire dans l'espace restant après distribution entre chaque item.

bloc 1
shrink
bloc 3
.item {

}

Flex-basis

Définit la taille initiale de l'élément avant que l'espace restant ne soit distribué aux autres items.

.item {
flex-basis: longeur /*ou*/ auto /*par défaut en mode auto. La longueur peut être en %, en px, ...*/;
}

Flex

Flex permet de déclarer en une seule ligne les valeurs de flex-grow, flex-shrink (optionnel) et flex-basis dans le CSS.

.item {
flex: none /*ou*/ valeur de flex-grow /*et*/ valeur de flex-shrink /*et*/ valeur de flex-basis;
}

Align-self

Align-self permet de gérer l'alignement d'un seul item indépendamment des autres. Ses propriétés sont les mêmes que celles d'align-items.

Accendente poenae quod.

Align Self

Ob haec aliis aliquando et quoque.

.item {
align-self: auto /*ou*/ flex-start /*ou*/ flex-end /*ou*/ center /*ou*/ baseline /*ou*/ stretch;
}
